# 홈택스 로그인 API

## 공동인증서 홈택스 로그인 & 홈택스 유저 정보 가져오기 스크래핑 API
### 사용 방법
- git clone
- cd htx
- app.js 파일에 유저, 세팅 정보 입력
- yarn start
- request : yourServer/login
- response : 홈택스 유저 정보를 확인하세요.
- 추후 인증서 등록 등 기능 추가 예정
